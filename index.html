Ship Iceberg Kaggle Competetion

A Kaggle competition where we were challenged to build an algorithm that automatically identifies if a remotely sensed target is a ship or iceberg. The data was in the form of an image, therefore I implemented RESNET (Residual Network) in keras, Multi Layer Perceptron (MLP) in tensorflow and convolutional neural network having initial layers of pre-trained VGG net in pytorch. Implemented various techniques like dropout, regularization, Batch Normalization to improve the accuracy of the model. Performed exploratory analysis by visualizing individual Convolution layers, weights, and gradients of layers using Tensorboard.
